# Week Calendar Uchi.ru

## Run backend

---

### Install packages

```
$ cd ./backend
$ npm install
```

### After installation finished

```
$ npm start
```

## Run frontend

---

### Install packages

```
$ cd ./frontend
$ npm install
```

### After installation finished

```
$ npm start
```
### Now open `http://localhost:3000/` and you'll see the calendar for the week
